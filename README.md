# openui5_heroku
Aplicação básica OpenUI5 para deploy no heroku


Baixar dependências
```
npm install
```

Iniciar em desenvolvimento
```
npm run dev
```

Iniciar construção 'self-contained'

```
npm run build
```

Iniciar projeto construído
```
npm run start
```
